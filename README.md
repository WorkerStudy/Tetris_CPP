### Управление:
- Стрелка Верх :arrow_up:
  - Вращает фигуру 
- Стрелка Вниз :arrow_down:
  - Опускает фигуру вниз 
- Стрелка Влево :arrow_left:\Вправо :arrow_right:
  - Перемещает фигуру по горизонтали
- ESC :x:
  - Выход
